Features

Everything you need.
Nothing you don't.

Built for developers, writers, and privacy advocates who want complete control over their writing tools.

Absolute Privacy

No databases, no accounts, no tracking. The local engine runs fully offline. AI requests use a stateless edge function — text is never stored.

Bring Your Own AI

Paste your API key for OpenAI, Groq, OpenRouter, Together, or Ollama. Pay fractions of a cent — directly to the provider.

Local-First, No Backend

On-device grammar needs no key or network. For AI, your key talks straight to the provider — or run a local model with Ollama. There is no server in between.

40+ Grammar Rules

Offline checking for passive voice, repetition, spelling, punctuation, clichés, redundant phrases, double negatives, and more.

Tone Rewriting

Rewrite text in 8 tones — Formal, Casual, Professional, Friendly, Concise, Detailed, Persuasive, and Neutral. Right-click or Ctrl+Shift+R.

Writing Statistics

Readability scores (Flesch, ARI), reading/speaking time, vocabulary diversity, and detailed analytics for every document.

Architecture

Dual-engine.
Maximum power.

Two engines work in parallel — fast local rules plus context-aware AI — then merge and rank results into a single clean list.

1

Local Rules Engine

40+ regex rules run instantly in your browser. Passive voice, spelling, punctuation, clichés — all offline, all free.

2

AI Processing

For complex grammar and context-aware suggestions, text is sent to your chosen LLM via a stateless edge function.

3

Smart UI

Red, amber, and blue wavy underlines appear as you type. Click to apply, ignore, or add to your dictionary.

Get Started

Ready in 5 minutes.

Clone, build, load. That's it.

# 1. Clone the repository
git clone https://github.com/swadhinbiswas/opengrammar.git

# 2. Build the extension
cd opengrammar/opengrammar/extension
bun install && bun run build

# 3. Load in chrome://extensions → Load Unpacked → dist/
#  Set your provider + API key in the popup. No backend needed.
