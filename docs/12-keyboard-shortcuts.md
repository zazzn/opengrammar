## 🔧 Customizing Shortcuts

### Chrome / Brave / Edge

1. **Open Extensions Shortcuts**
   ```
   chrome://extensions/shortcuts
   ```
   Or: `brave://extensions/shortcuts` / `edge://extensions/shortcuts`

2. **Find OpenGrammar**
   - Scroll to OpenGrammar in the list

3. **Edit Shortcut**
   - Click on the shortcut field
   - Press your desired key combination
   - Click **Save**

4. **Test Shortcut**
   - Try the new shortcut
   - Ensure it doesn't conflict with other extensions

### Firefox

1. **Open Add-ons**
   ```
   about:addons
   ```

2. **Open Shortcut Settings**
   - Click gear icon ⚙️
   - Select **Manage Extension Shortcuts**

3. **Edit OpenGrammar Shortcuts**
   - Find OpenGrammar
   - Click shortcut field
   - Press new combination
   - Changes save automatically

### Opera

1. **Open Extensions**
   ```
   opera://extensions/
   ```

2. **Enable Developer Mode**
   - Toggle in top-right

3. **Click "Keyboard Shortcuts"**
   - Find OpenGrammar
   - Edit shortcuts

## 💡 Shortcut Tips

### Best Practices

✅ **Choose memorable combinations:**
- Use action-related keys (R for Rewrite)
- Keep hands on home row
- Use modifier keys consistently

✅ **Avoid conflicts:**
- Don't use `Ctrl+C`, `Ctrl+V`, `Ctrl+Z`
- Avoid browser shortcuts (`Ctrl+T`, `Ctrl+W`)
- Check system shortcuts first

✅ **Ergonomic considerations:**
- Use one hand when possible
- Avoid awkward combinations
- Consider left-handed vs right-handed

### Recommended Custom Shortcuts

| Purpose | Windows/Linux | Mac |
|---------|---------------|-----|
| Rewrite | `Ctrl+Alt+R` | `Ctrl+Opt+R` |
| Statistics | `Ctrl+Alt+S` | `Ctrl+Opt+S` |
| Settings | `Ctrl+Alt+,` | `Ctrl+Opt+,` |
| Toggle | `Ctrl+Alt+E` | `Ctrl+Opt+E` |

## 🎯 Using Shortcuts Effectively

### Rewrite Workflow

1. **Select Text**
   - Click and drag to highlight
   - Or use `Shift+Arrow` keys

2. **Press Rewrite Shortcut**
   - `Ctrl+Shift+R` (Windows/Linux)
   - `Cmd+Shift+R` (Mac)

3. **Choose Tone**
   - Click tone button or use number keys
   - 1: Formal, 2: Casual, 3: Professional, etc.

4. **Apply or Copy**
   - `Enter` to apply
   - `Ctrl+C` to copy
   - `Esc` to close

### Quick Check Workflow

1. **Finish Writing**
   - Complete your text

2. **Manual Grammar Check**
   - Press `Ctrl+Shift+G`
   - Review all suggestions

3. **Apply Fixes**
   - Click suggestions to apply
   - Or use keyboard navigation

## ⚠️ Troubleshooting

### Shortcut Not Working

**Possible causes:**

1. **Conflict with browser shortcut**
   - Check `chrome://extensions/shortcuts`
   - Change to different combination

2. **Extension not loaded**
   - Reload extension
   - Check if enabled

3. **Wrong context**
   - Some shortcuts only work in text inputs
   - Try in different context

4. **Focus issue**
   - Click on webpage first
   - Ensure page has focus

### Shortcut Conflicts

**How to resolve:**

1. **Identify conflict**
   - Try shortcut
   - See what happens

2. **Check all shortcuts**
   - Browser: `chrome://settings/shortcuts`
   - Extensions: `chrome://extensions/shortcuts`
   - OS: System settings

3. **Change conflicting shortcut**
   - Pick unused combination
   - Test immediately

## 🔍 Advanced Shortcut Tips

### Power User Techniques

#### 1. Muscle Memory Training
- Practice same shortcut daily
- Use for one week consistently
- Becomes automatic

#### 2. Shortcut Chaining
Combine shortcuts for efficiency:
```
Select Text → Ctrl+Shift+R → Choose Tone → Enter → Done!
```

#### 3. Context-Aware Usage
- Different shortcuts for different contexts
- Email: Rewrite + Formal
- Chat: Quick check only
- Docs: Full statistics

#### 4. Custom Workflows
Create your own workflow:
```
Write → Check (Ctrl+Shift+G) → Rewrite (Ctrl+Shift+R) → Stats (Ctrl+Shift+S)
```
